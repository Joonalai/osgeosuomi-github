# OSGeo Suomi ry

This site is for hosting repositories of open source projects managed by OSGeo Suomi ry.

If you have any questions or problems related to the OSGeo Suomi organization page, please write an issue [for this repo](https://github.com/osgeosuomi/.github/issues) or send an email to info at osgeo.fi

## Policies

- Only OSGeo Suomi ry members can be a repo maintainer
- You can join the association at [website](https://www.osgeo.fi/jaseneksi/)
- Non-members may be normal contributors
- In case of problems, OSGeo Suomi ry board members have the right to remove repositories/code/rights under the organization (e.g. when the repository maintainer's user account is hijacked, etc.)
- Contributor guidelines: [CONTRIBUTING.md](../CONTRIBUTING.md)

### Creating repos

- Think it over carefully
- Add at least one other maintainer to the repo

tai lähetä sähköpostia info at osgeo.fi

## Politiikkaa

- Vain yhdistyksen jäsen voi olla repon ylläpitäjä (maintainer) 
  - Voit liittyä jäseneksi OSGeo Suomi ry:n [nettisivuilla](https://www.osgeo.fi/jaseneksi/)
- Muut voivat olla normaaleja jäseniä
- OSGeo Suomi ry:n hallituksen jäsenillä on oikeus poistaa organisaation alla olevia repositoryjä/koodia/oikeuksia ongelmatapauksissa esim repositoryn ylläpitäjän käyttäjätilin tullessa kaapatuksi yms
